Cara Install Laravel Di Windows

    1.Langkah Pertama Download dulu Aplikasi GitHUB/GitBash
  • 2.Buka Website Laravel lalu cari Documentation lalu pilih install with composer
  • 3.Setelah Selesai Download kemudian klik install saja gitbash nya di next nextkan
  • 4.Buka Folder Project laravel kemudian buka gitbash
  • 5.Pastekan Perintah berikut "composer create-project --prefer-dist laravel/laravel blog"
